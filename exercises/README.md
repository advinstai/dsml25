# Exercises

This directory contains hands-on exercises and tutorials for the school.

## Directory Structure

- **pandas/**: Data wrangling exercises with Pandas
- **matplotlib/**: Data visualization exercises
- **pytorch/**: Tensor operations and basic PyTorch exercises
- **neural-networks/**: Building and training basic neural networks
- **cnns/**: Convolutional Neural Networks exercises
- **rnns/**: Recurrent Neural Networks exercises
- **transformers/**: Transformer architecture exercises
- **applications/**: Domain-specific application exercises

## Exercise Format

Each exercise directory should contain:
- `README.md`: Overview and instructions
- `exercise_XX.ipynb`: Jupyter notebook with the exercise
- `solution_XX.ipynb`: Solution notebook (if applicable)
- `data/`: Any required datasets
- `requirements.txt`: Python dependencies for the exercise

## How to Use

1. Clone this repository
2. Navigate to the exercise directory
3. Install required dependencies: `pip install -r requirements.txt`
4. Open the Jupyter notebook: `jupyter notebook exercise_XX.ipynb`
5. Follow the instructions in the notebook

## Prerequisites

- Python 3.8+
- Jupyter Notebook or JupyterLab
- Basic knowledge of Python programming

## General Dependencies

Common libraries used across exercises:
- numpy
- pandas
- matplotlib
- pytorch
- scikit-learn

See individual exercise directories for specific requirements.
