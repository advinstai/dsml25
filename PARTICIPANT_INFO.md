## Software Setup

### Prerequisites

Before the school starts, please ensure you have:

1. **Python 3.8 or higher**
   - Download from: https://www.python.org/downloads/

2. **Jupyter Notebook or JupyterLab**
   ```bash
   pip install jupyter jupyterlab
   ```

3. **Required Libraries**
   ```bash
   pip install -r requirements.txt
   ```
   Or install individually:
   ```bash
   pip install numpy pandas matplotlib torch torchvision scikit-learn
   ```

4. **Text Editor or IDE** (optional but recommended)
   - VS Code: https://code.visualstudio.com/
   - PyCharm: https://www.jetbrains.com/pycharm/

### Verify Installation

Test your setup by running:
```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
import torch

print("NumPy version:", np.__version__)
print("Pandas version:", pd.__version__)
print("PyTorch version:", torch.__version__)
```

## Poster Presentation

### Requirements

- **Format**: Printed banner (NOT A4 or A3 paper)
- **Size**: Maximum 1m (width) x 1.5m (length)
- **Content**: Research description, methodology, results (if available)
- **Language**: English

### Presentation Schedule

Posters will be displayed and discussed during:
- Coffee breaks (10:30-11:00 and 15:30-16:00)
- Lunch intervals

### Tips for a Good Poster

- Clear title and author information
- Well-organized sections (Introduction, Methods, Results, Conclusions)
- Use figures and diagrams to illustrate concepts
- Keep text concise and readable from a distance
- Be prepared to discuss your work in 3-5 minutes
