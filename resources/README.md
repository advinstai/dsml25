## Categories

- **Pattern Recognition and Machine Learning** by Christopher Bishop

## Recommended Reading Order

For participants new to ML:

1. **Before the school**:
   - Python basics (if needed)
   - Introduction to NumPy and Pandas
   - Basic statistics review

2. **During the school**:
   - Follow lecture materials
   - Complete exercises
   - Explore related papers

3. **After the school**:
   - Deep dive into specific topics of interest
   - Implement projects
   - Read advanced papers
